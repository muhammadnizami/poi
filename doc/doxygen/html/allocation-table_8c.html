Detailed Description

mengatur seluruh senarai berkait alokasi berkas.

Author
jang berikoetnja Memperhitungkan juga ruang kosong. Bila ingin aman dengan perhitungan ruang kosong, panggil ini dan tidak langsung ke allocation-block-manager.h dalam hal manipulasi senarai. Setiap akhir manipulasi harus memanggil saveAllocTableCaches()
